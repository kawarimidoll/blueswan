# Blueswan

Chrome / Firefox extension for [Bluesky](https://bsky.app).

Feel your sky without annoying numbers.

https://chromewebstore.google.com/detail/blueswan/cnenmkcimleaeklmeflfnejicbhndlko
https://addons.mozilla.org/firefox/addon/blueswan/
